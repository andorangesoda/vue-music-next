# vue-music-next

基于Vue3+Webpack开发的Web App音乐软件。

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### 注意⚠️

1、`axios` 请使用 `^0.21.1`版本，不然可能无法获取到 song 的播放 url 。
